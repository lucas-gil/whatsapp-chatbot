# ⚡ QuickStart - 30 segundos

## Windows

```bat
REM 1. Instale Node.js: https://nodejs.org/

REM 2. Abra PowerShell e execute:
cd C:\Users\SEU_USUARIO\whatsapp-chatbot
npm install
npm run dev

REM 3. Abra no navegador:
REM Admin: http://localhost:3001
REM API: http://localhost:3000
```

## Mac / Linux

```bash
# 1. Instale Node.js
# Mac: brew install node
# Linux: sudo apt-get install nodejs npm

# 2. Configure
cd ~/whatsapp-chatbot
npm install
npm run dev

# 3. Abra no navegador
# Admin: http://localhost:3001
# API: http://localhost:3000
```

## Próximas Ações
